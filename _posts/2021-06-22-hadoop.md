# 2. 하둡 설치하기

root 홈 디렉터리로 온다

이 명령어로 간단히 다운로드한다.

```
wget http://apache.mirror.cdnetworks.com/hadoop/common/hadoop-3.3.0/hadoop-3.3.0.tar.gz
```

압축을 해제한다.

```
tar -zxvf hadoop-3.3.0.tar.gz
```

# 3. 하둡 설정하기

### 3-1. 환경변수 설정하기

환경변수를 설정해준다.

```
vim /etc/profile
```

파일안 맨밑에 

```
export JAVA_HOME=/usr/lib/jvm/java-1.8.0-openjdk-1.8.0.292.b10-1.el7_9.x86_64
export HADOOP_HOME=/root/hadoop
export PATH=$PATH:$JAVA_HOME/bin:$HADOOP_HOME/bin:$HADOOP_HOME/sbin
```

위 코드로 수정해준다.

저장하고 나와서 

```
source /etc/profile
```

최종적으로는 /etc/profile에 다음줄들이 추가되있다.

![image](https://user-images.githubusercontent.com/48349693/122897732-3ba96380-d385-11eb-94b5-ebd1666e0730.png)

HADOOP_HOME은 편의상이고

JAVA_HOME은 hadoop이 JAVA_HOME 환경변수를 참조하기때문에 필수이다.







hadoop jar $HADOOP_HOME/share/hadoop/mapreduce/hadoop-mapreduce-examples-3.3.0.jar wordcount test.txt wordcount_output
