### 1. 자바 설치하기

root로 바꾼다.

자바가 깔려있긴 하지만 jps 명령어가 없으므로 다시 설치한다.

이 명령어로 간단히 설치 한다.

yum install java-1.8.0-openjdk-devel

![h1](https://user-images.githubusercontent.com/48349693/122888696-0a2c9a00-d37d-11eb-87d3-3ce5f7c9ce73.PNG)

이 경로가 새로 설치한 자바경로이다. 여기의 bin 폴더에 jps명령어가 있다.

```
/usr/lib/jvm/java-1.8.0-openjdk-1.8.0.292.b10-1.el7_9.x86_64
```

jps 명령어가 안될 경우

/etc/profile 을 vim 으로 열어서 맨밑에 아래코드를 넣어준다.

```
export JAVA_HOME=/usr/lib/jvm/java-1.8.0-openjdk-1.8.0.292.b10-1.el7_9.x86_64
export PATH=$PATH:$JAVA_HOME/bin
```
