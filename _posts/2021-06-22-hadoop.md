# 3. 하둡 설정하기

### 3-1. 환경변수 설정하기

환경변수를 설정해준다.

```python
vim /etc/profile
```

파일안 맨밑에 

```python
export JAVA_HOME=/usr/lib/jvm/java-1.8.0-openjdk-1.8.0.292.b10-1.el7_9.x86_64
export HADOOP_HOME=/root/hadoop
export PATH=$PATH:$JAVA_HOME/bin:$HADOOP_HOME/bin:$HADOOP_HOME/sbin
```

위 코드로 수정해준다.

저장하고 나와서 

```python
source /etc/profile
```

최종적으로는 /etc/profile에 다음줄들이 추가되있다.

![image](https://user-images.githubusercontent.com/48349693/122897732-3ba96380-d385-11eb-94b5-ebd1666e0730.png)

HADOOP_HOME은 편의상이고

JAVA_HOME은 hadoop이 JAVA_HOME 환경변수를 참조하기때문에 필수이다.

---

### 3-2. ssh 키 생성하기

root 홈 디렉터리로 온다.

```python
ssh-keygen
```

위 명령어를 실행한다. ssh-keygen은 엔터만 친다.

그리고 아래 명령어를 실행한다.

(아래 명령어는 나중에 서버(slave)를 추가할 때 사용한다. 지금은 master서버를 다루고 있다.)

```python
ssh-copy-id -i ~/.ssh/id_rsa.pub root@localhost
```

그러면 ssh root@(주소) 를 쳤을때 비밀번호 없이 해당 클라이언트에 접속할 수 있다.

---

### 3-3. 설정파일 수정하기

현재 디렉토리를 변경한다.

```python
cd ~/hadoop/etc/hadoop
```

<span style="color:blue">hadoop-env.sh</span>

해당 파일을 연다.

```python
vim hadoop-env.sh
```

54번줄 주석을 풀고 JAVA_HOME을 수정한다.

```python
export JAVA_HOME=/usr/lib/jvm/java-1.8.0-openjdk-1.8.0.292.b10-1.el7_9.x86_64
```

<span style="color:blue">core-site.xml</span>

해당파일을 다음과 같이 수정한다.

```python
<?xml version="1.0" encoding="UTF-8"?>
<?xml-stylesheet type="text/xsl" href="configuration.xsl"?>

<configuration>
  <property>
    <name>fs.defaultFS</name>
    <value>hdfs://192.168.127.132:9000</value>
  </property>
</configuration>
```

ip 주소 부분은 꼭 master 서버의 주소를 넣어야한다. 

나중에 master서버를 복사할 것이기 때문에 localhost로 쓰면 안된다.

<span style="color:blue">hdfs-site.xml</span>

해당파일을 다음과 같이 수정한다.

```python
<?xml version="1.0" encoding="UTF-8"?>
<?xml-stylesheet type="text/xsl" href="configuration.xsl"?>

<configuration>
  <property>
    <name>dfs.replication</name>
    <value>1</value>
  </property>
</configuration>
```

숫자 1은 파일 복제 수 이다.

여러개의 서버가 있을때 보통은 3을 사용한다.
